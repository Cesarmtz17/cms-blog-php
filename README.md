# CMS Blog

Sistema de gestión de contenido para blogs con panel de administración, editor WYSIWYG y gestión de categorías.

## Características
- Panel de administración
- Editor CKEditor 5
- Gestión de posts y categorías
- Autenticación de usuarios

## Tecnologías
- PHP 7+
- MySQL + Medoo
- Semantic UI
- jQuery

## Instalar
1. Configurar base de datos en `res/php/init.php`
2. Subir a servidor con PHP y MySQL
